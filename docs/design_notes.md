# Design Notes

## トーン
- 硬派な信頼感を軸にする
- 華やかさはレイヤー、透明感、余白、写真演出で加える
- 過剰なエンタメ感は避ける

## 配色方針
- Primary: Navy 系
- Secondary: 淡い Sky Blue
- Accent: Orange
- 背景は濃紺ベースに明度差をつけ、情報の階層を明確にする

## UI方針
- ナビは glassmorphism
- 各セクションは full-width band 構成
- 繰り返し要素は card UI で整理
- hover は軽い浮き上がりと境界線の変化に限定
- アニメーションは fade / translate のみ

## OCM寄りにするポイント
- 情報を生活者目線で分離整理する
- カルチャー、福利厚生、研修、キャリアを混在させない
- 温度感は少し上げるが、言い切りは控えめにする

## 実装メモ
- Tailwind CDN で完結
- インラインの最小JSでスクロールフェードのみ実装
- 画像はプレースホルダーを使い、後で差し替え可能な構造にする
