# e-gravity.co.jp Renewal Project Brief

## プロジェクト概要
- 対象: e-gravity.co.jp コーポレートサイト刷新
- 担当範囲: 実装、ドキュメント更新
- 前提: 技術領域、事業領域、働き方の実態は変えず、見せ方のみ進化させる

## リニューアル方針
- 8割Gravity、2割OCM
- 硬派な技術企業としての信頼感は維持する
- UI、情報整理、見せ方、コピーの温度感を改善する
- 華やかさは演出として加えるが、会社実態の誇張には使わない

## 表現ルール
- 技術軽視に見える表現は避ける
- 過剰にポップな配色や装飾は避ける
- コピーは落ち着きと解像度を優先する
- OCMは構造設計のみ参考にし、文言模倣は行わない

## 必須メッセージ
- 製造業の現場に向き合う会社であること
- 組込・制御を軸とした技術力があること
- 長期で着実に力がつく環境であること
- カルチャー、福利厚生、研修、キャリアが整理されていること

## 実装要件
- HTML5 セマンティック構造
- Tailwind CSS CDN 利用
- モバイルファースト
- 軽量構成
- JavaScript は最小限
